技术支持

如果你在使用鸭压时遇到问题，可以先参考下面的说明；如果仍未解决，欢迎通过 GitHub Issues 反馈。

常见问题

为什么部分视频需要先从 iCloud 下载？

如果原始文件当前只保存在 iCloud，鸭压会先请求下载原视频，然后再开始压缩或预览处理。

为什么不同排序方式下列表顺序会变化？

鸭压支持按体积、视频码率、拍摄时间和添加时间进行浏览，切换排序后会按新的维度重排列表。

压缩后原视频会自动删除吗？

不会。鸭压会将压缩结果写回系统相册，删除或回收原视频必须由你主动确认执行。

GitHub Issues： https://github.com/Borry5/yaya-site/issues
